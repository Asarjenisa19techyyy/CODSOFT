# Titanic Prediction System 🚢

An advanced Machine Learning project that predicts Titanic passenger survival using Random Forest Classification and passenger insights.

## Features
- Survival prediction system
- Passenger analysis
- Data visualization graphs
- Custom passenger testing
- Machine Learning model

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
Random Forest Classifier

## Model Accuracy
Achieved approximately 80% prediction accuracy.
